Setup Instructions
New Outlook & Outlook on the Web
  1. Fill in your details, then click Copy for Outlook & Web.
  2. In Outlook, go to Settings → Accounts → Signatures → New signature.
  3. Name it (e.g. SLG Standard), click into the body and press Enter twice to leave two blank lines above the signature.
  4. Paste with Ctrl + V, then set it as the default for new messages (and replies, if you wish).
  5. Click Save.
Classic Outlook (Windows Desktop)
  1. Quick method — click Copy for Outlook & Web, then in Classic Outlook go to File → Options → Mail → Signatures, create a new signature, paste with Ctrl + V, and save. This renders well, with only very minor spacing differences.
  2. Exact method (recommended) — for pixel-perfect results, use the file instead:
    1. In Signatures, create a new signature named SLG Standard, type a placeholder character, and click OK. This registers the signature in your profile.
    2. Click Download for Classic Outlook above and note where the .htm file saves.
    3. Fully close Outlook.
    4. Open File Explorer, paste %APPDATA%\Microsoft\Signatures into the address bar, and press Enter.
    5. Delete SLG Standard.htm, then rename the downloaded file to SLG Standard.htm and move it into this folder. Leave the .rtf and .txt files as they are.
    6. Reopen Outlook and set SLG Standard as the default for new messages.
Outlook Mobile App (iPhone & Android)
  1. Open this tool on your phone's browser and fill in your details. (Mobile signatures are separate from desktop — they don't sync automatically.)
  2. Tap Copy Mobile Signature.
  3. In the Outlook app, tap your profile icon → the gear (Settings) → Signature.
  4. Select and delete any existing text, then paste and save.
The mobile signature uses a correctly-sized logo and icons, so nothing blows up to full width the way a pasted desktop signature does. If your app strips images entirely, the text lines still read cleanly on their own.
Outlook for Mac (macOS)
  1. Click Copy for Outlook & Web.
  2. In Outlook, go to Outlook → Settings → Signatures and click +.
  3. Name it, paste into the edit area with Cmd + V, and close the window to save.
  4. Under Choose default signature, assign it to your account for New messages.
